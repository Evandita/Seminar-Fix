REFERENSI TAMBAHAN UNTUK ITERATIVE PEELING DAN CLIENT-SIDE MEMORY

ITERATIVE PEELING (TERINSPIRASI DARI GRAPH THEORY):

[ref12] Bollobás, B., & Riordan, O. (2004). "The Diameter of a Scale-Free Random Graph". Combinatorica, 24(1), 5-34. DOI: 10.1007/s00493-004-0002-2
(Catatan: Artikel ini membahas peeling algorithms dalam konteks graph decomposition dan analisis struktur graf kompleks)

[ref13] Batagelj, V., & Zaveršnik, M. (2011). "Fast Algorithms for Determining (Generalized) Core Groups in Social Networks". Advances in Data Analysis and Classification, 5(2), 129-145. DOI: 10.1007/s11634-010-0079-y
(Catatan: Artikel ini menjelaskan k-core decomposition dan peeling process untuk menganalisis struktur berlapis dalam graf)

CLIENT-SIDE MEMORY (TERINSPIRASI DARI WEB DEVELOPMENT):

[ref14] Hickson, I. (Ed.). (2011). "Web Storage (Second Edition)". W3C Recommendation. World Wide Web Consortium. URL: https://www.w3.org/TR/webstorage/
(Catatan: Spesifikasi resmi W3C tentang localStorage dan sessionStorage API untuk penyimpanan data di sisi klien pada web browser)

[ref15] Zakas, N. C. (2012). "Professional JavaScript for Web Developers" (3rd ed.). Wrox Press. ISBN: 978-1118026694, Chapter 23: "Offline Applications and Client-Side Storage"
(Catatan: Buku referensi standar tentang client-side storage technologies dalam web development, termasuk localStorage, IndexedDB, dan application cache)
